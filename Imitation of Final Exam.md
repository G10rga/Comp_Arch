# ✅ მოკლე პასუხების გასაღები

**ვარიანტი 1**
- 1ა) >2ტბ მყარი, 128 განყოფ., Secure Boot, GUI+მაუსი, ქსელი/ენები, UEFI დრაივ., სწრაფი ჩატვირთვა. 1ბ) ორჯერ ფრონტზე გადაცემა → 2×, დაბალი ძაბვა, დიდი მოცულობა, მაღ. სიხშირე. 1გ) Hi-Color 2¹⁵=32768; Real Color 2¹⁶=65536 ფერი.
- 2ა) მიმდევრობითი → ნაკლები გამტარი, მაღ. სიჩქარე, ცხელი ჩართვა, წვრილი კაბელი. 2ბ) OTG — მოწყობ. ერთმანეთს დაუკავშირდეს host-ის გარეშე (ID კონტაქტი). 2გ) Power Good — ადასტურებს ძაბვების ნორმას, პროცესორს რესეტიდან უშვებს.
- 3ა) Matrix Storage — RAID 0/1 ერთ ან რამდენ დისკზე. 3ბ) SMART — თვითდიაგნოსტიკა, მტყუნების პროგნოზი. 3გ) CD-R — ერთჯერადი, საღებავი ფენა იწვის ლაზერით.
- 4ა) Single-ended — 1 სიგნ.+საერთო მიწა, იაფი, ხმაურისადმი მგრძნ. 4ბ) DRAM იაფი/დიდი/რეგენერაცია→ ძირ. RAM; SRAM სწრაფი/ძვირი→Cache. 4გ) POST, Setup, ჩამტვირთავი, დაბ. დონის დრაივერები.
- 🧮 5ა) 120/(12×0.8)=**12.5 ა**. 5ბ) PCI პარალელ./საერთო სალტე; PCIe მიმდევრ./დუპლექს/ტრასები (×1..×16). 5გ) ტექსტური — ეკრანი სიმბოლოების მატრიცა, ASCII+ატრიბუტი.

**ვარიანტი 2**
- 1ა) ROM→PROM→EPROM(UV)→EEPROM/Flash(გადაწერადი). 1ბ) DDR3: 8n-prefetch, 1.5ვ, მეტი სიხშირე/მოცულობა. 1გ) 1.0/2.0→8b/10b; 3-5→128b/130b; 6-7→242b/256b.
- 2ა) NTFS — ჟურნალირება, კვოტები, დაშიფვრა, კომპრესია, უფლებები. 2ბ) USB3 5გბიტ/წმ, 900მა, +2 წყვილი, full-duplex. 2გ) HD Audio (Azalia) — მრავალარხ., 192კჰც, AC'97-ზე უკეთესი.
- 3ა) დიფ. — პირდაპირი+ინვერსიული, სხვაობა იკითხება, ხმაურმედეგი, მაღ. სიჩქარე. 3ბ) Power_On — ჩართვა/გამორთვა; +5VSB — მუდმივი მცირე კვება (Wake). 3გ) 20-კონტ.: +3.3/+5/+12/-12/-5ვ, PG, +5VSB, Power_On.
- 4ა) BIOS — საბაზო I/O პროგრამები ROM-ში; AMI, Award, Phoenix. 4ბ) SATA-1 150მბ/წმ, SATA-2 300, SATA-3 600. 4გ) CD-RW — გადაწერადი, ფაზურ-ცვალებადი ფენა.
- 🧮 5ა) 3200/16=**200 მჰც**. 5ბ) Fast Memory Access — კონტროლერი ამცირებს მიმართვის დაყოვნებას. 5გ) რეცირკ. ფილტრი — ჰერმეტულ შიგნით მტვერს ფილტრავს.

**ვარიანტი 3**
